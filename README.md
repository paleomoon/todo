实现了一个 ToDo 的单页应用。

使用 localStorage 存储数据，JSON 模拟数据表，实现了任务分类、编辑、改变状态等功能，具有良好的交互体验。  

优化重构及工程化：  
- 使用SASS重构CSS代码，使用嵌套、继承、混合器等方式，使代码更加简洁。
- 使用require.js模块化javascript代码，降低了耦合性。
